# ContactsUtils
A small Ruby gem for **validation** and **normalization** of email addresses and phone numbers.

More information about validation and normalization of emails and phone numbers can be found here:
- [email validation](https://whoapi.com/blog/understanding-valid-email-address-formats-a-short-guide/)
- [email normalization](https://www.reddit.com/r/webdev/comments/1i1i84x/email_address_normalization_what_is_the_best/)
- [phone validation](https://www.activityinfo.org/support/docs/forms/validating-phone-numbers-with-regular-expressions.html)
- [phone normalization](https://www.draft-docs.miarec.com/admin-guide/system/phone-number-normalization/)
